# Backend Project

This is a backend project built using Node.js, Express, and MongoDB.

---

## 🚀 Features

- User Authentication (Register/Login)
- Password hashing using bcrypt
- JWT authentication
- REST APIs
- MongoDB database integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
